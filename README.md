Приложение для применения средств механистической интерпретируемости к LLM.

В данный момент реализованы:
- Карты активаций
- Аблации
- Атрибуция логитов
- Логитные линзы
- Патчинг активаций

<p>Пользователи имеют возможность создавать проекты - наиболее крупные сущности, обладающие некоторой целью, описанием и определяющие модель, над которой ведётся работа.</p>
<p>В каждом проекте имеются эксперименты - менее крупные сущности, обладающие более сжатыми целями.</p>
<p>Внутри каждый эксперимент представляет собой цепочку ячеек - текстовых и модельных. </p>
<p>В текстовых ячейках пользователи могут заносить информацию.</p>
<p>В модельных ячейках производится взаимодействие с ML моделями, с сохранением истории применения инструментов и вызова объектов из истории для повторного отображения результатов.</p>

Применяется архитектура отдельных сервисов.
Сервисы:
- auth.py - аутентификация пользователей (JWT токен)
- model.py - сервис каталога доступных моделей
- user.py - сервис с пользователями и проектами
- orchestrator.py - сервис обработки запросов взаимодействия с моделями
